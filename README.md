QuotesReady — UK Construction Estimating Platform

Role: Freelance Full-Stack Developer
Client: Simon Whittle, Founder — QuotesReady
Stack: Base44 · VAPI (Voice AI) · Stripe · Google OAuth · DNS/SEO


The Problem

UK construction contractors lose time producing accurate quotes for clients  manual estimating is slow, inconsistent, and pulls tradespeople away from the field. Simon needed a platform that lets contractors generate professional estimates quickly, including a hands-free option for use on-site.

What I'm Building

An end-to-end estimating SaaS platform, built in close collaboration with the founder:


Voice Assistant Integration ("Mia")  a VAPI-powered voice assistant that lets contractors generate and manage quotes hands-free, ideal for use on a job site where typing isn't practical.
Stripe Billing  subscription and payment infrastructure for the platform's paid tiers.
Google OAuth  secure, low-friction sign-in for contractors.
Production Infrastructure  full DNS configuration and SEO setup to get the platform properly indexed and live for UK search traffic.
Ongoing Production Support  ongoing bug resolution and stability work as the platform scales toward launch.


Approach

Working directly with the founder in a fast, iterative cycle — shipping features, verifying them live in production (not just trusting platform-reported status), and adjusting based on real usage as the contractor base grows.

Status

Actively in development, with core voice assistant, billing, and auth flows in place and production infrastructure live.


This is a case study for a proprietary client project  source code is not public due to client confidentiality. Happy to walk through the voice assistant flow or demo the platform on a call.
